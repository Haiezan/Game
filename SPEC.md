# 飞行躲避小游戏 - 规范文档

## 1. 项目概述
- **项目名称**: 星际穿梭
- **类型**: 竖屏俯视飞行躲避游戏
- **核心玩法**: 玩家控制飞船在太空中穿梭，躲避随机出现的障碍物，收集能量晶体获得分数
- **目标用户**: 休闲游戏玩家

## 2. 视觉与渲染规范

### 场景设置
- **视角**: 2D竖屏俯视
- **画布尺寸**: 400x600px，居中显示
- **背景**: 深空星空，星星闪烁效果

### 视觉风格
- **整体风格**: 复古像素风 + 霓虹发光效果
- **主色调**: 深蓝背景 (#0a0a1a)，青色飞船 (#00ffff)，红色障碍物 (#ff3366)，黄色能量 (#ffdd00)
- **字体**: "Press Start 2P" (Google Fonts) 像素风格字体

### 效果
- 飞船引擎尾焰动画
- 碰撞爆炸粒子效果
- 能量收集闪光效果

## 3. 游戏机制

### 飞船
- 尺寸: 40x30px
- 初始位置: 屏幕底部中央
- 移动速度: 6px/帧
- 移动范围: 限制在画布内

### 障碍物
- 类型: 陨石 (不规则圆形)
- 尺寸: 20-50px随机
- 生成位置: 屏幕顶部随机X位置
- 下落速度: 3-6px/帧，随分数增加
- 生成间隔: 每60帧生成一个

### 能量晶体
- 尺寸: 15x15px
- 外观: 闪烁的黄色菱形
- 出现概率: 30% (与障碍物同时生成)
- 分数: +10分

### 分数系统
- 存活时间: 每帧 +1分
- 收集能量: +10分
- 碰撞障碍物: 游戏结束

## 4. 交互规范

### 控制方式
- 键盘 ↑/↓/←/→ 控制移动
- 空格键开始/重新开始
- 触摸支持: 屏幕左右两侧Tap控制

### UI元素
- 左上角: 当前分数
- 顶部中央: 最高分记录
- 游戏结束: 显示最终分数和重新开始提示

## 5. 验收标准

- [ ] 游戏画布正确显示，居中布局
- [ ] 飞船可以用键盘控制移动
- [ ] 障碍物从顶部随机下落
- [ ] 能量晶体随机出现可收集
- [ ] 碰撞检测正常工作
- [ ] 分数实时显示并记录最高分
- [ ] 游戏结束后可重新开始
- [ ] 星空背景动画效果
- [ ] 飞船有尾焰效果
