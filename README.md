# Game

HTML5 游戏集合

## 游戏列表

| 游戏 | 类型 | 操作 |
|------|------|------|
| [星际穿梭](星际穿梭.html) | 飞行躲避 | 方向键移动 |
| [将士大战](将士大战.html) | 塔防策略 | 收集能量放置士兵 |
| [火柴人大战](火柴人大战.html) | 动作对战 | 方向键 + 空格 |

## 运行方式

直接在浏览器中打开 `index.html` 进入游戏大厅，或打开任意游戏 HTML 文件直接游玩。

## 控制说明

- **方向键** - 移动
- **空格键/Z键** - 开始/攻击
- **触摸支持** - 屏幕左右两侧点击

## 技术栈

- 原生 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- Google Fonts (Press Start 2P)
